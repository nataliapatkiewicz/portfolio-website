June's Journey

Senior Environment Artist — Wooga
Environments Mobile Hidden Object

Goal

Design richly detailed environments for a hidden-object game — scenes that tell a story while concealing dozens of interactive items. Must work beautifully at mobile resolution.

Process

Developed a layered composition system — foreground, midground, background — with color temperature and lighting to guide the eye. Each scene went through 3–4 rounds of gameplay testing.

Outcome

Contributed 20+ scenes across 3 seasonal updates. Player engagement metrics for these scenes exceeded the team average by 15%. Approach was adopted as the studio standard.

Environment Design

Evening Terrace — The hero scene for the Autumn update. Warm lighting draws the eye through a layered 1920s setting while concealing 35+ interactive objects across three depth planes.

Mediterranean Set

Greek Taverna — Saturated palette shift for the Summer chapter. Bold complementary colors (blue doors, terracotta, yellow accents) create focal points for gameplay.

Daytime Scene

Coastal Dock — Cooler tonal range to contrast warmer interior scenes. Open composition with strong horizon line provides visual rest between dense gameplay moments.
