Editorial Editorial Section Layouts

These go inside <div class="project-editorial"> and are revealed when the user clicks "View more". Mix and match freely within a single project.

Ultrawide

ed-ultrawide

Full-width image spanning edge to edge, text card below matching the same width. Best for panoramic art, wide landscapes, and hero moments.

Medium

ed-medium

Image takes 2/3, text card takes 1/3. Best for landscape art (like June's Journey environments). Add flip class to swap sides.

CARD (1/3)

Flipped: ed-medium flip = card left, image right

Slim

ed-slim

Two portrait/phone-ratio images side by side + text card. Best for mobile UI, phone screenshots, character lineups. Use card-first or card-last to position the card.

Tip: Image descriptions

Each ed-img can have an optional hover description. Add a <div class="img-desc"> inside with a <span class="desc-label"> and a <p>. It slides up on hover. You can also omit it entirely for images that don't need explanation.

Folder structure

Each project lives in assets/projects/project-name/. Keep all images for one project in one folder. When talking to Claude, just say "add a new ultrawide section using ultra-01.jpg from project-name" and it'll know exactly what to do.

Example folder

assets/projects/junes-journey/
  preview-01.jpg      // hero mosaic - evening terrace
  preview-02.jpg      // hero mosaic - greek taverna
  preview-03.jpg      // hero mosaic - coastal dock
  ultra-porch.jpg     // editorial ultrawide section
  med-office.jpg      // editorial medium section
  med-grayscale.jpg   // editorial medium flipped section
  slim-ui-01.jpg      // editorial slim section
  slim-ui-02.jpg      // editorial slim section

Workflow Adding a New Project

Step-by-step process for adding a complete project with editorial sections.

Create folder & name images

Create assets/projects/your-project/ and add your images using the naming conventions above.

Copy the blank template

In index.html, find the "BLANK TEMPLATE" comment near the bottom of the work section. Copy everything between the CUT lines.

Fill in project details

Replace PROJECT NAME, role, tags, and image paths. Choose a preview grid layout (layout-hero, layout-pair, etc.).

Build editorial sections

Inside <div class="project-editorial">, add ed-ultrawide, ed-medium, and/or ed-slim sections. Mix types to create an editorial flow. You can have as many or as few as you want.

Remove what you don't need

Not every project needs all 3 editorial types. Delete entire ed-section blocks you don't need. You can also omit the <div class="project-editorial"> entirely if a project should only have the preview grid.

Talking to Claude

Use these phrases for fast edits: "Add an ed-ultrawide section to June's Journey using ultra-01.jpg" or "Flip the medium section in Book of Quack" or "Add a new ed-slim with slim-01.jpg and slim-02.jpg to the casual game project". Claude will know exactly which component and files you mean.
